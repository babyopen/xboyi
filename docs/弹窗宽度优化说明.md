# 生肖预测详情弹窗宽度优化

## 📋 优化概述

调整了生肖预测详情弹窗的宽度，使其更加紧凑和美观。

---

## 🔧 修改内容

### 1. 桌面端宽度调整

**之前：**
```css
.modal-content {
  width: 90%;
  max-width: 25rem;  /* 400px */
}
```

**现在：**
```css
.modal-content {
  width: 85%;
  max-width: 22rem;  /* 352px */
}
```

**变化：**
- 宽度从 `90%` 减小到 `85%`
- 最大宽度从 `25rem (400px)` 减小到 `22rem (352px)`
- 减少了 **48px** 的最大宽度

---

### 2. 移动端宽度调整（≤767px）

**之前：**
```css
@media (max-width: 767px) {
  .modal-content {
    width: 95%;
    max-width: none;  /* 无限制 */
  }
}
```

**现在：**
```css
@media (max-width: 767px) {
  .modal-content {
    width: 90%;
    max-width: 20rem;  /* 320px */
  }
}
```

**变化：**
- 宽度从 `95%` 减小到 `90%`
- 添加了最大宽度限制 `20rem (320px)`
- 避免在小屏幕上过宽

## ✅ 优化优势

### 1. **视觉平衡**
- 弹窗不再占据过多屏幕空间
- 两侧留白更合理，视觉更舒适

### 2. **内容可读性**
- 两列布局（统计信息 + 评分）更加紧凑
- 文字不会拉伸过长，阅读体验更好

### 3. **移动端适配**
- 添加了最大宽度限制
- 避免在平板等设备上弹窗过宽

### 4. **一致性**
- 与其他弹窗组件保持一致的宽度规范
- 符合iOS设计规范（推荐最大宽度 320-375px）

---

## 🎨 设计参考

### iOS Human Interface Guidelines

根据苹果的设计规范：
- **iPhone**: 弹窗宽度通常为屏幕宽度的 85-90%
- **iPad**: 弹窗最大宽度建议为 375px
- **Mac**: 弹窗最大宽度建议为 400-500px

我们的调整：
- 桌面端：最大 352px（接近 iPad 标准）
- 移动端：最大 320px（接近 iPhone SE 宽度）
